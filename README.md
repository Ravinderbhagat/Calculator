A clean, iOS-inspired calculator built with vanilla HTML, CSS, and JavaScript — no libraries, no frameworks.
Features

Basic arithmetic: addition, subtraction, multiplication, division
Live expression display (shows the full equation as you type)
Keyboard support (0–9, operators, Enter, Backspace, Escape)
Sign toggle (+/−) and percentage (%) functions
Chained operations without pressing equals
Division by zero handled gracefully (shows Error)
Responsive dark UI with smooth button animations

Tech Used: HTML5, CSS3, JavaScript (ES6)
Live Demo: https://ravinderbhagat.github.io/Calculator/
